# 排行榜类型：ACM

ACM风格的排行榜。但与传统的ACM排行榜具有如下差别：

1. 可以设置每题的分数
2. 对于编译错误，也算作罚时
3. 罚时计算方式为：`(提交次数-1)*提交罚时+最后一次提交的相对时间`

## 配置

```ini
type=basic
# Topstar数，设置为0则不显示
topstars=10
# 罚时计算的参考开始时间，必须设置
startTime=08/17/1926, 8:00:00 PM
# 每次额外提交的罚时，单位为毫秒。默认为20分钟
penaltyPerSolution=1200000
```
